<h1>Streamlit Data Dashboard Project</h1>


<h2>Description</h2>
This project is an interactive data visualization dashboard built with Streamlit. I completed the full workflow—setting up the environment, performing exploratory data analysis (EDA), building visualizations, and deploying the application using Render. The dashboard loads a CSV dataset and allows users to explore it through interactive charts and UI components.<br><br>

<h2>Project Overview</h2>

- Create a Streamlit-based web application
- Perform exploratory data analysis in Jupyter Notebook
- Visualize the dataset through histograms and scatterplots
- Add interactive components such as checkboxes
- Deploy the live dashboard using Render

<h2>Languages and Utilities Used</h2>

- Python

- scikit-learn

  - Logistic Regression
  - Gradient Boosting Classifier
  - XG Boost Classifier
  - Light Gradient Boosting Machine Classifier

<h2>Objective:</h2>
Predict whether a customer will churn, defined as:<br>
if end_date = "No" → Customer is active<br>
if end_date = a date → Customer has churned
