<h2>App Webpage</h2>

On the sidebar, users can select both the vehicle condition and vehicle manufacturer. These selections dynamically feed into a groupby() operation, which calculates the average price for each vehicle type based on the chosen filters. The resulting bar chart updates in real time, allowing users to explore how condition and make influence the pricing of different vehicle categories. <br/>
